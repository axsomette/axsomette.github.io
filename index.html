Comment ça marche ?

Bienvenue dans MonBudget, votre assistant de gestion financière nouvelle génération. Suivez les étapes ci-dessous pour maîtriser votre budget en toute simplicité.

1. Ajouter une opération

Utilisez le formulaire pour ajouter une dépense ou un revenu. Saisissez le nom, le montant, puis sélectionnez le Mois et l'Année correspondants via les menus déroulants.

2. Gérer les Catégories & Objectifs

Cliquez sur "Catégories" en haut. Vous pouvez créer des grands "Dossiers" (ex: Maison) et des sous-catégories à l'intérieur.

💡 Budget Max : Vous pouvez définir un Budget Max pour chaque grand dossier. Une barre de progression interactive s'affichera alors pour vous aider à respecter cet objectif !

3. Simulateur de Budget

Le bouton "Simulateur" dans le menu du haut est votre nouvel assistant prévisionnel. Cliquez dessus, choisissez le mois de référence sur lequel vous souhaitez vous baser, indiquez le budget maximal souhaité pour le mois prochain, et l'algorithme calculera automatiquement les catégories exactes où vous devez faire des efforts pour atteindre cet objectif !

4. Analyses Intelligentes & Thème

Bilan & Analyses : Sélectionnez un mois précis pour voir des conseils générés automatiquement. Si vous sélectionnez "Tout l'historique", vous accédez désormais à un Bilan Global de vos finances depuis toujours.

Mode Sombre : Cliquez sur la petite icône pour basculer sur le thème Premium sombre.

5. Sauvegarder & Importer

Vos informations sont stockées temporairement dans votre navigateur. Pour ne rien perdre :

Sauvegarder : Télécharge un fichier de secours (.json) contenant toutes vos données. À faire à chaque fois !

Importer : Chargez un fichier pour Fusionner ou Écraser (remplacer vos données existantes).
